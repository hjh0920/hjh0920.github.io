round (四舍五入取整)

Y = round(X,N) 四舍五入到 N 位数：

  • N > 0：舍入到小数点右侧的第 N 位数。
  • N = 0：四舍五入到最接近的整数（默认）。
  • N < 0：舍入到小数点左侧的第 N 位数。
